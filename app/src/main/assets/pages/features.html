FEATURES

  • SEARCH: This is one of the best feature offered in this application. With this feature, a user can search for any Legal Maxim, Landmark Judgment, or a word contained in any Chapter of any act.
  • NIGHT MODE: This feature enables a user to change the background of the text from white to black. This feature can be used at night, to avoid arduous/strenuous reading.
  • REFERENCE NOTES: This feature enables a user to create his own short notes in the application itself. With this feature, a user can feed his own interpretation to any word/ sentence contained in any section, Legal Maxim or Landmark judgment and can have a quick reference in future.
  • TEXT HIGHLIGHT: This feature enables a user to highlight any word/ sentence contained in any section, Legal Maxim, Landmark judgment for quick future reference.
  • DICTIONARY: This is one of the best feature offered in the application. This feature enables a user to search the meaning of any word contained in any section instantly without closing the application and going in any browser.
  • JUMP: This is one of the best feature provided in this application. With this feature, a user can directly jump from one act to another act without coming back to the home screen and read the concerned section/provision in the other act and then return to the previous act.
  • SHARE: With this feature, a user can share a section with any person.
  • ZOOM-IN and ZOOM-OUT: With this feature, a user can change the size of the text as per his choice.
  • BOOKMARK: This feature enables a user to bookmark any legal maxim, landmark judgement or a chapter contained in any act.
